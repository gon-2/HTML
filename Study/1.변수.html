변수

  let, const 
      let   : 최초에 변수 선언할때 let을 이용해 선언하면 변수의 중복 사용을 방지할 수 있다.
      const : 절대로 바뀌지 않는 상수(수정 x), 대문자로 선언하는것이 약속
      변하지 않는 값을 const, 변할수 있는 값은 let으로 선언
      

      1.  변수는 문자와 숫자, $와 _만 사용할 수 있다.
      2. 첫글자는 숫자가 될 수 없다.
      3. 예약어는 사용할 수 없다.
      4. 상수는 대문자로 작성한다.
      5. 변수명은 읽기쉽고 이해할 수 있게 선언 해야한다.

자료형

     문자형 변수는 3가지로 작성 할 수 있다. 
     1. '' : let name = 'jhon';
     2. "" : let name = "jhon";
     3. `` : let message = `My name is ${name}`;
                 message = `나는 ${30+1}살 입니다.`; 식의 표현식도 사용가능 

     1, 2번과 큰 차이가 없음
     3. ``은 문자열 내부의 변수를 표현해줄때 사용하면 편리하다

숫자형

     let age = 30; // 숫자형 Number
     const PI = 3.14;

     숫자형은 사칙연산이 가능함
     +, -, *, /, %(더하기, 빼기, 곱하기, 나누기, 나머지)

     이때 let x = 1/0; 이처럼 숫자를 0으로 나눈다면 Infinity(무한대의 값이 나온다)
     이때 let y = name/2; 이처럼 문자열을 숫자로 나눈다면 NaN(Not a number: 숫자가 아니다) 라는 값이 나온다.

     숫자형 변수 작업을 한다면 console로 NaN이 나오는지 잘 확인해야함

논리형

    // Boolean
    let a = true; //참
    let b = flase; // 거짓

    let name = 'jhon';
    let age = 30;

    console.dir(name == 'mike'); // true
    console.dir(age > 40);       // false

null, undefined

    let age; // 이처럼 선언만 하고 값을 넣지않은 상태로 호출을 하면 undefined가 뜬다
    let user = null; // 이처럼 변수에 null을 넣는다면 user는 존재하지 않는다고 생각하면 된다

typof 연산자

    let name = "jhon";

    console.dir(typeof 3);          // number
    console.dir(typeof name);       // string
    console.dir(typeof true);       // boolean
    console.dir(typeof "xxx");      // string
    console.dir(typeof null);       // object
    console.dir(typeof undefined);  // undefined

    * 이렇게 각 타입을 확인 할 수 있다.
    * 자신이 선언한 변수는 타입을 쉽게 알 수 있지만 다른 개발자가 선언한 변수의 타입을 알아야 하거나
    * API통신을 통해 받아온 데이터를 타입에 따라 다른 방식으로 처리해야할때 사용한다.
